#Mi sitio web

## Dev
1- Clona el repositorio
2- Instala las dependencias con `npm install`
3- Clonar el archivo `.env.template` a `.env` y configurar las variables de entorno necesarias.
4- Ejecutar `npx prisma migrate dev`
5- Iniciar el servidor de desarrollo con npm run dev
6- Ejecutar el seed con `npm run seed`
